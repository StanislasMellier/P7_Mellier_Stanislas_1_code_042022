# Creation d'un réseau social d’entreprise dans le cadre de la formation Développeur Web par OpenClassrooms

Ce projet utilise React comment framework frontend ,express pour le backend et l'Api , mysql comme base de données.

## Installation

Afin d'installer l'application vous devez cd dans les dossier frontend et backend et executer la commande `npm i` puis dans le dossier frontend vous devez executer `npm run start` et dans le dossier backend vous devez executer `npm run dev`
